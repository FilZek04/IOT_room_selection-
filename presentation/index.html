Database Design — Why MongoDB

  • Schema flexibility — sensor types have different fields; document model avoids rigid migrations
  • Time-series fit — compound index on (room, type, timestamp) for fast range queries
  • Async driver — Motor integrates natively with FastAPI's async request handling

AHP — How It Works

  • Pairwise comparisons on Saaty 1–9 scale — intuitive for users
  • Eigenvector method extracts priority weights from comparison matrices
  • Consistency Ratio (CR < 0.1) ensures logical coherence of preferences
  • Weighted Sum aggregates criteria scores using hierarchy weights
  • Two-phase: hard facility constraints → AHP scoring

5 Modular Components

  • pairwise_matrix.py — Matrix construction
  • eigenvector.py — Weight extraction + CR
  • score_mapping.py — Sensor → [0,1]
  • aggregation.py — WSM / WPM options
  • ahp_engine.py — Orchestrator
